How to use

Use the "TabMenu" Essence component to render a "TabItem" component. The "TabMenu" Essence component will return a div container with class name "snackbar". Inside the TabMenu component you can include only the TabItem component.

• Structure 

<TabMenu>
  <TabItem type='list' id='tab-1' label='Tab 1' />
  <TabItem type='content' id='tab-1'>
    can be used to include any component or html code
  <TabItem>

  <TabItem type='list' id='tab-2' label='Tab 2' />
  <TabItem type='content' id='tab-2'>
    can be used to include any component or html code
  <TabItem>
</TabMenu>
• Styling
- use the TabItem type='content' component to customise any elements with styles from Styles > Colors menu


• Options
- id: required to set the id attribute for the TabItem component
- type: available options are 'list' for the tab navigation and 'content' for the tab content
- label: required to set the TabItem type='list' navigation text
- use the examples from above to structure your own TabMenu
