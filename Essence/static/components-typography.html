Typography

Typography is such an important part of a catchy website, so you must use it carefully. You are just a few classes away of doing it! To do that, we are here helping you with predefined classes.

Text Alignment

Aligning text with Essence is annoyingly simple. Just use the .e-text class with -left, -right, -center and -justify in the paragraph class. Yeah, it’s that simple: .e-text-center

Text Transformation

You need all text to be uppercase, lowercase or capitalize. We know, it happens. So we’ve created easy-to-use classes to do just that.

Uppercase Text

capitalize text

lowercase text

REACT CODE

        
<Text type='p' 
  classes={"e-text-uppercase"}
  >
  UPPERCASE TEXT
</Text>

<Text type='p' 
  classes={"e-text-capitalize"}
  >
  Capitalize Text
</Text>

<Text type='p' 
  classes={"e-text-lowercase"}
  >
  lowercase text
</Text>

ESSENCE NATIVE CODE

        
<Text style={[
  Typography['e-text-uppercase'], 
]}>
  UPPERCASE TEXT
</Text>
<Text style={[
  Typography['e-text-capitalize'], 
]}>
  Capitalize Text
</Text>
       
<Text style={[
  Typography['e-text-lowercase'], 
]}>
  lowercase text
</Text>
