Dividers

According to our needs we created some dividers and stored them in a .less file called, you've guest it, dividers.

We've created three values for width and five values for height. You can combine them as you please.

For example, to get a thin, long divider you add the classes .e-divider .thin .long to an empty <div> and set it's background color.

To get a divider to strech the full width of its parent container, simply leave out the width value as in the following example: .e-divider .thick

The available values for height are: thinnest, thin, regular, thick, thickest. For width we have: short, medium, long.

REACT CODE

          
<Divider classes={'thin long e-background-indigo-700'} />

ESSENCE NATIVE CODE

          
 <Divider style={[
   Helpers['thinnest'],
   Helpers['medium'],
   Helpers['e-margin-top-50'],
   Helpers['e-margin-bottom-50'],
   Colors['e-background-grey-400'],
   ]} />

HTML CODE

          
<div class="e-divider thin long e-background-indigo-700"><div>

How to use

The "Divider" Essence component will return a div container with the class name "e-divider" to separate content.

• Structure 

<Divider classes={'__css-classes__'} />
• Styling
- classes: thick - for minimum height of 5px or thin for minimum height of 1px

How to use - Essence Native

The "Divider" Essence component will return a View component with styling from style array.

• Structure 

<Divider 
  style={[__HelpersStyles__, __ColorsStyles__]} 
/>
• Styling
- style: add styling class names to View component from Styles > Colors menu
