Tooltips

Tooltips are labels that appear on hover and focus when the user hovers over an element with the cursor, focuses on an element using a keyboard (usually through the tab key), or, in a touch UI, upon touch (without releasing). Check out some examples and how to use them below.

HTML CODE

          
<button class="e-btn-default raised" data-tooltip="Tooltip text">default</button>

<button class="e-btn-primary raised" data-tooltip="Tooltip text">primary</button>

<button class="e-btn-succes raised" data-tooltip="Tooltip text">succes</button>

<button class="e-btn-info raised" data-tooltip="Tooltip text">info</button>

<button class="e-btn-warning raised" data-tooltip="Tooltip text" data-position="down">warning</button>

<button class="e-btn-danger raised" data-tooltip="Tooltip text" data-position="up">danger</button>

How to use

Use the "Btn" Essence component to render a group of "BtnItem" components. The "Btn" Essence component will return a div container with a group of <button> html tags. Inside the Btn component you can include only the BtnItem component.
A "BtnItem" Essence component will render one < button > html tag.

• Styling
- classes: add extra class names to BtnItem components from Styles > Colors menu

• Options
- tooltipText: generates a tooltip message a the bottom of the element
- tooltipPosition: position the tooltip message - top / bottom / right / left
