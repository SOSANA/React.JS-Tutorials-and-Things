How to use

Use the "SnackBar" Essence component to render a "SnackBarItem" component. The "SnackBar" Essence component will return a div container with class name "snackbar". Inside the SnackBar component you can include only the SnackBarItem component.

Use the "Toast" Essence component to render a "ToastItem" component. The "Toast" Essence component tag will return a div container with class name "toast". Inside the Toast component you can include only the ToastItem component.

• Structure 

<SnackBar>
  <SnackbarItem id={'snackbar_id'}>
    can be used to include any component or html code
  </SnackbarItem>
</SnackBar>


<Toast>
  <ToastItem id={'toast_id'}>
    can be used to include any component or html code
  </ToastItem>
</Toast>
• Styling
- classes: add extra class names to any Snackbar or Toast component or subcomponent from Styles > Colors menu


• Options
- id: required for unique Snackbar or Toast to be shown on click
- in order to recreate the examples above please use a BtnItem component with the options:
• snackbar='snackbar_id' where snackbar_id is the SnackBar id option
• toast='toast_id' where toast_id is the Toast id option
- use the examples from above to structure your own SnackBar or Toast
