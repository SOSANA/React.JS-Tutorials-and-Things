In material design, every pixel drawn by an application resides on a sheet of paper. Paper has a flat background color and can be sized to serve a variety of purposes. A typical layout is composed of multiple sheets of paper. Here you have some examples in the most common shapes and states.

How to use

Use the "Paper" tag to render a group or single "PaperItem" component. The "Paper" React tag will return a div container with a single or multiple <PaperItem> React tags with the class name "e-paper".

• Structure 

<Paper>
  <PaperItem>
    __react-components__
  </PaperItem>
  <PaperItem type={'circle'}>
    __react-components__
  </PaperItem>
  <PaperItem type={'sharp'}>
    __react-components__
  </PaperItem>
</Paper>
• Styling
- classes: add extra class names to Paper or PaperItem component from Styles > Colors menu

• Options
- type
1. circle: append "circle" to class names list with the use of PaperItem component
2. sharp: append "sharp" to class names list with the use of PaperItem component
- use the examples from above to structure your own List

How to use - Essence Native

Use the "PaperItem" Essence component to render a View iOS component as container with Text iOS component as a child

• Structure 

<PaperItem>
  text or string
</PaperItem>

<PaperItem type={'circle'}>
  text or string
</PaperItem>

<PaperItem type={'sharp'}>
  text or string
</PaperItem>
• Styling
- containerStyle: add styling class names to View component from Styles > Colors menu
- textStyle: add styling class names to Text component from Styles > Colors menu

• Options
- onPress callback function is called on press
- type:
  1. circle: used to style the container of View component
  2. sharp: used to style the container of View component
- use the examples from above to structure your own List
