Bottom sheets:

You know those cool looking sheets that pop-up from the bottom of the screen? Yeah, we have those too. Depending on how you want information to be displayed, aligned or arranged, use the appropriate code. Here are some examples to get you started.

How to use

Use the "BottomSheets" Essence component to render a group of "BottomSheetsItem" components. The "BottomSheets" Essence component will return a div container with class name "e-bottom-sheet". Inside the BottomSheets component you can include only the BottomSheetsItem component.
A "BottomSheetsItem" Essence component will render one List component.

• Structure 

<BottomSheets>
  <BottomSheetsItem>
    can be used to include List component
  </BottomSheetsItem>
</BottomSheets>
• Styling
- use the List Styling in order to create your own styles

• Options
- id: required to work with a Btn component
- in order to recreate the example above please use a BtnItem component with the options bellow, where actionClick is the BottomSheet event, actionType.action to show the BottomSheetsItem div and actionType.targetID is the BottomSheetsItem id option :
• actionClick='bottomsheets' and
• actionType={{
    'action': 'show',
    'targetID': 'bottomsheet-simple'
}}
