Get Started with Essence Native

Learn how to create your iOS app

Essence and React Native

Essence Native is built on top of React Native. That means you can create your mobile application writing javascript, but using native device functions. At the moment, React Native is only available for iOS apps, but soon will be working on Android devices too.

To start using Essence Native you must meet some minimum requirements:
  • OS X and Xcode, as for the moment it only runs on Mac.
  • Homebrew is the recommended way to install node, watchman, and flow.
  • Next, run this in the terminal: brew install node
  • brew install watchman - so you won't hit a node file watching bug

When you have met these requirements, you can start developing your iOS application.
  • npm install -g react-native-cli will install react native globally on your system.
  • Then initialize the app using whatever name you want for your application: react-native init MyApplication.

Right now you are ready to install Essence, to use all the predefined and pre-styled components for your product.
  • npm install react-native-essence --save

Now open your index.ios.js file and require Essence. Right after the line that requires 'react-native' paste the following code line: var Essence = require('react-native-essence');.

To use the components you need from Essence, you must create a variable that extends the components from Essence. The following code contains all the components, but you can remove the ones you don't need. 
var {
      Utils, Colors, Helpers, Typography, 
      Buttons, GetStarted, BtnItem, Divider, Progress, PaperItem, Card, SwitchItem, SliderItem, DialogItem, DatePicker, Menu, List,
} = Essence;

That's it! The documentation and examples for each component and style helper can be found in their own presentation pages. Use the left menu to navigate through them.
