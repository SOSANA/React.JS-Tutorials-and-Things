Buttons

There are 3 types of material buttons: FAB (floating action button), Raised and Flat.

FAB Button

We believe FAB stands for fabulous. Here’s a fabulous class for it: .e-btn-fab.
The small one has the "-mini" addition to the class.

REACT CODE

        
<Btn>
  <BtnItem
    icon='action-favorite'
    type='fab'
    rippleEffect={true}
  />
  <BtnItem
    icon='action-favorite'
    type='fab'
    rippleEffect={true}
    mini={true}
  />
</Btn>

ESSENCE NATIVE CODE

        
<BtnItem
  containerStyle={[
    Colors['e-background-pink-500'], 
    Buttons['e-btn-fab]
  ]} 
  textStyle={Colors['e-text-white']}>
  Fab
/>
       
<BtnItem
  containerStyle={[
    Colors['e-background-pink-500'], 
    Buttons['e-btn-fab-mini]
  ]} 
  textStyle={Colors['e-text-white']}>
  Fab
/>

How to use

Use the "Btn" Essence component to render a group of "BtnItem" components. The "Btn" Essence component tag will return a div container with a group of <button> html tags. Inside the Btn component you can include only the BtnItem component.
A "BtnItem" Essence component will render one < button > html tag.

• Structure 

<Btn>
  <BtnItem
    classes={'flat'}
    label='Label Default'
    type='default'
    rippleEffect={true}
  />
</Btn>
• Styling
- classes: add extra class names to BtnItem components from Styles > Colors menu
- icon: the name of icon from Styles > Icons (generate a i html tag with the class name "e-icon-" + icon)
- type: appends a string (available: default, fab, primary, succes, info, warning, danger) to the class name "e-btn-" in the following format "e-btn-" + type

• Options
- rippleEffect: true or false (generate a ripple Effect)
- mini: appends the e-btn-mini to the class names
- name: name of the button
- submit: if set to true it changes the type button attribute from button to submit, the default type is button
- label: generates the element button text
- tooltipText: generates a tooltip message a the bottom of the element
- tooltipPosition: position the tooltip message - top / bottom / right / left
- disabled: if set to true it uses the "disabled" button attribute
- actionClick: any value from the following list: navigation, bottomsheets, datepicker or dialog
- actionType: has the following options:
-   action: 'show' to display a component or 'hide' to hide a component
-   id: any string that represent the 'id' option for a component

How to use - Essence Native

Use the "BtnItem" component to render a View iOS native container with a Text iOS native component inside.
A "BtnItem" Essence component will render native iOS View and Text components.

• Structure 

  <BtnItem
    containerStyle={[__ButtonStyles__]}
    textStyle={[__ColorsStyles__]}>
      Button Label
  </BtnItem>
• Styling
- containerStyle: add styling class names to View component from Styles > Colors menu
- textStyle: add styling class names to Text component from Styles > Colors menu

• Options
- onPress: callback function is called on press
