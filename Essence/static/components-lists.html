Lists

Lists are best suited to presenting a homogeneous data type or sets of data types, such as images and text. Take a look at the examples below and pick one - and keep it consistent as much as possible.

How to use

Use the "List" Essence component to render a group or single "ListItem" component. The "List" Essence component will return a div container with a single or multiple <ListItem> Essence components with the class name "e-list".

• Structure 

<List>
  <ListItem>
    __li-style-block-or-react-components__
  </ListItem>
</List>
• Styling
- classes: add extra class names to List or ListItem component from Styles > Colors menu

• Options
- type: single-line (for a single line text), two-line (for a two lines text) or multi-line (for 3 lines text)
- use the examples from above to structure your own List

How to use - Essence Native

Use the "List" Essence component to render a View component with custom content including "TouchableHighlight", "View" or "Text" native iOS components.
Each child from inside of List component will be rendered as a View native component.

• Structure 

<List style={[__HelpersStyles__]}>
  <TouchableHighlight />,
  <View /> or
  <Text />
</List>
• Styling
- style: add styling class names to List component from Styles > Colors menu

• Options
- no options

• Content
- can be use together with "TouchableHighlight", "View" or "Text" native iOS components
