How to use

Use the "Navigation" Essence component to render a "List" component. The "Navigation" Essence component will return a aside container with class name "e-nav-drawer". Inside the Navigation component you can include only the List component.
A "List" Essence component will render any List subcomponent or html code.

• Structure 

  <Navigation>
    <List classes='e-list-navigation' id='NavigationDrawer'>
      <ListItem listType='navigation' contentText='navigation 1' contentLink='#navigation1' hasSubmenu={true} >
        <Block type='li'>
          <Text> List 1 </Text>
        </Block>
        <Block type='li'>
          <Text> List 2 </Text>
        </Block>
      </ListItem>
      can be used to include any component or html code
    </List>
  </Navigation>
• Styling
- classes: add extra class names to any Navigation component or subcomponent from Styles > Colors menu


• Options
- live: default true and if set to false will generate a inline style css with position: relative as in example above
- header: text for Navigation header
- footer: text for Navigation footer
- logo: is the image logo included in the Navigation header
- use the examples from above to structure your own Navigation
