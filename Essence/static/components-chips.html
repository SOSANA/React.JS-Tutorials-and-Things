How to use

Chip components are used in the search results to display contact information.
Use the "Chip" Essence component to render a group or single "ChipItem" components. The "Chip" Essence component will return a <ul> list container with a single or multiple <li> tag which contains a div with class "e-chip".

• Structure 

<Chip>
  <ChipItem
    image='__image-url__'
    imageAlt='__image-alt-text__'
    name='__contact-text-or-name__'
    email='__contact-email-address__'
  >
  </ChipItem>
</Chip>
• Styling
- classes: add extra class names to any ChipItem subcomponent from Styles > Colors menu

• Options
- image: image file for contact avatar
- imageAlt: image alternative text for contact avatar
- name: text for contact name
- email: email address for contact name
