Colors

Colors are pretty. And Essence has them all, powered by Google.

Background color:

To apply a background color, just add the color name and shade value value to the element as a class: .e-background-green-400‚Äù

REACT CODE

        
<Block type='div' 
 classes={"brick brick-12 e-background-red-100"}>
</Block>

ESSENCE NATIVE CODE

        
<View style={[
    Colors['e-background-red-100'],
]}>

Text color:

Essence

To set the text color, add the color and shade value to the element as a class, similar to the way background-color works: .e-text-red-400

REACT CODE

          
<Text type='h2' classes={"e-text-white"}>Text here</Text>

ESSENCE NATIVE CODE

          
<Text style={[
  Colors['e-text-white'],
]}>
