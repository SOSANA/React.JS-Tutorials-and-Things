Cards

There are several types of cards which can be used in different situations. These range from informational panels to prompts and single choice ones. Take a look at the examples to see how you can use them.

Image card with description and links

Going to Ibiza

Limit supplemental actions to two actions, in addition to an overflow menu.

HTML CODE

        
<div class="card" >
  <div class="card-main-image">
    <img src="assets/img/card-img.jpg" alt="Card Image" >
    <h1 class="e-headline">
      <span>Going to Ibiza</span>
      <span class="blur-effect" style="background-image:url(assets/img/card-img.jpg);"></span>
    </h1>
  </div>
  <div class="card-supporting-text">
    <h4 >Limit supplemental actions to two actions, in addition to an overflow menu.</h4>
  </div>
  <div >
    <button class="flat e-btn-default" ><span >No</span>
    </button>
    <button class="flat e-btn-danger" ><span >Yes</span>
    </button>
  </div>
</div>

REACT CODE

        
<Card>
  <CardItem>
    <CardItemHeader blur={true}>
      <Block type="div" classes={"card-main-image"}>
        <Image src="assets/img/card-img.jpg" alt="Card Header Image" />
        <Text type="h1" classes={"e-headline"}>
          Going to Ibiza
          <Block classes={"blur-effect"} image="assets/img/card-img.jpg" />
        </Text>
      </Block>
    </CardItemHeader>

    <CardItemContent classes={"card-supporting-text e-text-blue-grey-400"}>
      <Text type="h4">
        Limit supplemental actions to two actions, in addition to an overflow menu.
      </Text>
    </CardItemContent>

    <CardItemFooter>
      <Block type="div" classes={"e-left"}>
        <BtnItem
          classes={'flat e-text-indigo-400'}
          label='Yes'
          type='danger'
          rippleEffect={true}
        />
      </Block>
      <Block type="div" classes={"e-left"}>
        <BtnItem
          classes={'flat'}
          label='No'
          type='default'
          rippleEffect={true}
        />
      </Block>
    </CardItemFooter>
  </CardItem>
</Card>

How to use

Use the "Card" Essence component to render a group or single "CardItem" components. The "Card" Essence component tag will return a div container with a group or single <div> html tag with the class name "e-card". Inside the CardItem component you can include any component.

• Structure 

<Card>
  <CardItem>
    <CardItemHeader>
      Header: can be used to include Image and/or Text
    </CardItemHeader>
    <CardItemContent>
      Content: can be used to include any component
    </CardItemContent>
    <CardItemFooter>
      Footer: used for actions from single or multiple BtnItem items
    </CardItemFooter>
  </CardItem>
</Card>
• Styling
- classes: add extra class names to any Card subcomponent from Styles > Colors menu

• Options
- blur: if set to true it will render CardItemHeader without class name "card-header"
- the blur effect must be structured like the first example using a Block component with image option set to image path, e.g.:
<Block classes={"blur-effect"} image="assets/img/card-img.jpg" />
- use the examples from above to structure your own Card
