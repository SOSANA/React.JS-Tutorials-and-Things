</DatePicker>

How to use

Use the "DatePicker" Essence component to render the structure composed from DatePickerHeader, DatePickerContent and DatePickerFooter components. The "DatePicker" Essence component will return a div container with a single <div> html tag with the class name "e-picker-container".

We recomend to use a InputItem component with an actionClick='datepicker' and actionType.action: 'show' and actionType.id: '__datepicker-id__' }}

• Structure 

<DatePicker>
  <DatePickerHeader>
  <DatePickerContent>
  <DatePickerFooter>
    Footer: used for actions from single or multiple BtnItem items
  </DatePickerFooter>
</DatePicker>
• Styling
- classes: add extra class names to any DatePicker, DatePickerHeader, DatePickerContent and DatePickerFooter component from Styles > Colors menu

• Options
- id: required for unique DatePicker - use the examples from above to structure your own DatePicker

How to use - Essence Native

Use the "DatePicker" component to render the native iOS DatePicker.

• Structure 

<DatePicker style={[__ColorsStyles__]} />
• Styling
- style: add styling class names to any DatePicker component from Styles > Colors menu
