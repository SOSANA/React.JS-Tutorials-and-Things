List Controls

These indicate the state of a list item, information about a list item, or serve as an action related to the list item.

How to use

Use the "List" tag to render a group or single "ListItem" component. The "List" Essence component will return a div container with a single or multiple <ListItem> Essence components with the class name "e-list".

• Structure 

<List type='checkbox_or_switch' position='right'>
  <ListItem>
    __li-style-block-or-react-components__
  </ListItem>
</List>
• Styling
- classes: add extra class names to List or ListItem component from Styles > Colors menu

• Options
- type
1. checkbox: append "has-checkbox" to class names list with the use of SwitchItem component with type = checkbox
2. switch: append "has-switches" to class names list with the use of SwitchItem component with type = switch
- position: together with type: checkbox will position the checkbox on right side of the ListItem - use the examples from above to structure your own List

How to use - Essence Native

Use the "List" Essence component to render a View component with custom content including "TouchableHighlight", "View" or "Text" native iOS components.
Each child from inside of List component will be rendered as a View native component.

• Structure 

<List style={[__HelpersStyles__]}>
  <View />,
  <Text /> and
  <SwitchItem />
</List>
• Styling
- style: add styling class names to List, View or SwitchItem component from Styles > Colors menu

• Options
- no options

• Content
- can be use together with "View", "Text" native iOS components and "SwitchItem" Essence Native component
