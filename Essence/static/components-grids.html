ESSENCE NATIVE CODE

          
Essence native uses the flex layout, 
so grids are no loger needed.

How to use

Use the "Block" tag to render a group or single Essence components. The "Block" Essence component will return a div container with a single or multiple <div> html tags with the class name "brick".

• Structure 

<Block classes={'brick brick-12'}>
  __react-components-or-html-code__
</Block>
• Styling
- create a column based layout with "brick" class name, available columns: 1 to 12 - classes: add extra class names to any Block component or subcomponent from Styles > Colors menu
